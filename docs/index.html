📚 Data Sources

GLOBOCAN 2022 — IARC / WHO
Bray F et al. "Global cancer statistics 2022." CA: A Cancer Journal for Clinicians, 2024. Возрастно-стандартизованные показатели смертности и заболеваемости для 185 стран, 36 видов рака. Эталонный источник для онкологической эпидемиологии.
🫁 Рак лёгких 🟤 Рак желудка 🔴 Рак шейки матки ☀ Меланома 🟡 Рак поджелудочной Заболеваемость
🔗 gco.iarc.who.int · IARC/WHO · Peer-reviewed · 2024
WHO Global Health Observatory (GHO)
Основной источник данных. ВОЗ собирает статистику здоровья от министерств здравоохранения 194 стран. Данные проходят стандартизацию и нормализацию для сравнения между странами. Факторы риска: стандартизованная распространённость курения и ожирения, потребление алкоголя, загрязнение воздуха PM2.5, индекс охвата UHC.
Смертность от рака DALYs Инфраструктура 🚬 Курение ⚖ Ожирение 🍷 Алкоголь 💨 PM2.5 🏥 UHC Index
🔗 ghoapi.azureedge.net · Обновляется ежегодно · Бесплатно
ClinicalTrials.gov (NIH)
Реестр клинических исследований Национального института здоровья США. Содержит данные о всех зарегистрированных испытаниях лекарств и методов лечения по всему миру. Данные актуальны на дату последнего обновления карты.
Клинические исследования Фазы испытаний Активные протоколы
🔗 clinicaltrials.gov · Open API · Обновляется еженедельно
EUROCARE-5 / CONCORD-3
Крупнейшие европейские и глобальные исследования выживаемости онкологических пациентов. CONCORD-3 охватывает 71 страну, 37.5 млн пациентов. Данные — публичные сводные показатели из peer-reviewed публикаций.
5-летняя выживаемость Рак груди Рак лёгкого Рак простаты
🔗 The Lancet, 2018 · Peer-reviewed
Natural Earth / World Atlas TopoJSON
Географические данные карты мира. 177 стран в формате TopoJSON, разрешение 110m (оптимально для мировой карты). Используется NYT, Guardian, Bloomberg.
Границы стран Геометрии
🔗 github.com/topojson/world-atlas · Public Domain
